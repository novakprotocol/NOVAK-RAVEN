# Security Policy

**Contact:** licensing@novakprotocol.net  
**PGP:** (to be published with release)

## Reporting a Vulnerability
If you discover a potential vulnerability or cryptographic weakness in the NOVAK Protocol,
please report it privately to `security@novakprotocol.net`.  
Public disclosure before coordinated fix is discouraged.

## Supported Versions
Only tagged releases (v1.x) receive coordinated disclosure responses.

## Verification
Each ledger release is signed with Falcon-512 signatures;
Merkle roots are published in `docs/release-roots/`.
